Structure d'un fichier .pad


--------------------------------
title # titre du projet => égal au nom du fichier si non renseigné dans le GUI
n_couches # nombre de couches
nom1 nom2 nom3 etc...  # nom des couches des couches sous forme de liste séparée par un espace /// type = string
e1 e2 e3 etc... # épaisseurs des couches sous forme de liste séparée par un espace /// la derniere épaisseurs ne sert à rien /// type = float
E1 E2 E3 etc... # module des couches sous forme de liste séparée par un espace  /// type = float
n1 n2 n3 etc... # poisson des couches sous forme de liste séparée par un espace  /// type = float
i1 i2 i3 etc... # interfaces des couches sous forme de liste séparée par un espace /// la derniere interface ne sert à rien /// 0 = glissante , 1 = collée
